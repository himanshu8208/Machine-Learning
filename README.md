# 🏠 House Price Prediction System

## 📌 Overview
This project is a Machine Learning-based system that predicts house prices based on features like area, number of bedrooms, and bathrooms.

It uses **Linear Regression** to build a predictive model trained on housing data.

---

## 🚀 Features
- Predict house prices using ML
- Data preprocessing & cleaning
- Model training and evaluation
- Visualization of results

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📂 Dataset
- CSV dataset containing housing details
- Features:
  - Area (sq ft)
  - Bedrooms
  - Bathrooms
- Target:
  - Price

---

## ⚙️ How It Works
1. Load dataset
2. Clean and preprocess data
3. Split into training & testing sets
4. Train Linear Regression model
5. Predict house prices
6. Evaluate model performance

---

## 📊 Model Evaluation
- Mean Absolute Error (MAE)
- R² Score
